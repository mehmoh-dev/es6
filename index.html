What is ES6?

ECMAScript 6, also called ES2015, introduced modern JavaScript features that work especially well with React. These language improvements help you write cleaner components, manage data more safely, and avoid repeated code.

Note: ES6 is now standard in modern React development. Learning it will make React easier to use.

Core ES6 features used in React

Arrow functions
Shorter function syntax for components and event handlers.
Const and let
Use const for values that do not change and let for variables that do.
Destructuring
Extract values from props or objects with fewer lines of code.
Spread syntax
Copy array or object data without changing the original values.
Template literals
Create strings with embedded expressions and readable formatting.
Modules
Import and export components using a clear, reusable structure.

    Why this code is beginner-friendly

    • Each component is small and focused.
    • Props are passed clearly to child components.
    • The state update uses an immutable pattern with spread syntax.
    • Event handlers use arrow functions for easier context.

    What to remember

    • const is best for values that do not change.
    • Use destructuring to access props like {'{ id, text, completed }'}.
    • Spread syntax {'{ ...todo }'} copies object values safely.
    • JSX looks like HTML but uses JavaScript expressions inside curly braces.

    Beginner tips

    • Start by writing one component at a time.
    • Use console.log() when you want to inspect props or state.
    • Check your browser console for errors when JSX does not compile.
    • Remember that every component must return a single root element.
